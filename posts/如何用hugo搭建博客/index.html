如何用hugo搭建博客

June 17, 2020

注：

进入Hugo官网 点击 Quick Start 可以查看本文前七步骤的原文和源代码

步骤一：安装Hugo

Mac用户在终端运行

brew install hugo
hugo version

windows用户进入hugo发布页面，下载hugo_extended_xxxx_Windows-64bit.zip，解压，再将装有hugo.exe的路径添加到系统环境变量中，最后打开终端运行hugo version

步骤二

在终端创建一个空目录输入hugo new site quickstart ‘quickstart’ 替换成 ‘github用户名.github.io-creator’ 按回车 会生成一个 github用户名.github.io-creator 的目录

步骤三

 code github用户名.github.io-creator 目录
 git init(添加本地仓库)
 git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke (下载主题)
 echo 'theme = "ananke"' >> config.toml(配置主题)

步骤四

hugo new posts/文件名.md

文章写好后把 draft 后面的 true 改成 false

步骤五

hugo server -D

步骤六

code config.toml 看见如下代码，标题、语言、主题 都在这里设置，最后要将baseURL中的内容替换为自己 GitHub Pages 显示的网址

baseURL = "https://example.org/"
languageCode = "en-us"
title = "My New Hugo Site"
theme = "ananke"

步骤七

hugo -D (会创建一个新的目录名叫public)

步骤八

在 github用户名.github.io-creator 目录下创建
.ignore 文件，在里面添加/public/

cd public 
git init
git add .
git commit -v

在GitHub里创建一个新的远程仓库，名称是
github用户名.github.io

git remote add origin git@github.com:用户名/用户名小写.github.io.git
git push -u origin master

步骤九

进入GitHub 里仓库的 settings，找到 GitHub Pages 一栏，里面就有我们的个人博客地址。

额外补充

如何创建第二篇博客

cd 用户名.github.io-creator 目录

运行 hugo new posts/第二篇博客.md 编辑文章后

运行hugo -D

运行

git add . 
git commit -m xxxx
git push -f

在hugo中添加图片

将图片放到 static 目录下 然后输入 ![](图片路径) 即可
